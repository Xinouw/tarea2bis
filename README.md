


Este repositorio contiene dos programas en Python que:

1. **Generan** datos sintéticos de meteoros observados durante una ventana de 2 días.
2. **Analizan** esos datos para calcular el intervalo promedio de tiempo entre meteoros.

---

## Archivos principales

### `generate_meteors.py`

Script que genera los archivos de meteoros:

- Elige aleatoriamente una cantidad de eventos entre **500 y 999**.
- Para cada evento crea un archivo de texto llamado:

  ```text
  meteor_001.txt, meteor_002.txt, ..., meteor_XXX.txt
